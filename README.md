# 🏦 Final Project of Bank Management System 2018

An Excel-based Banking Management System developed for maintaining bank records, customer accounts, deposits, withdrawals, and generating reports automatically. This project demonstrates spreadsheet-based database management concepts using Microsoft Excel.

---

## 📖 Project Overview

**Final Project of Bank Management System 2018** is a banking information management system implemented in Microsoft Excel. The workbook is designed to manage customer accounts, transaction records, and generate reports for efficient data analysis.

The system demonstrates how spreadsheet tools and formulas can be used to simulate basic banking operations without requiring a dedicated database management system.

---

## ✨ Features

* Maintain bank and branch information.
* Store account details and balances.
* Record deposit transactions.
* Record withdrawal transactions.
* Generate reports for accounts, deposits, and withdrawals.
* Provide a switchboard menu for easy navigation.
* Calculate account balances automatically.
* Organize banking records efficiently.

---

## 📂 Workbook Structure

### 1. Main Sheet

Acts as the home page of the workbook.

### 2. Switch Board

Provides navigation to different modules of the system.

### 3. Bank Info

Stores bank information including:

* Bank ID
* Bank Name
* Branch Name
* Branch Code
* Phone Number

#### Example Banks

* HBL
* UBL
* ABL
* SBL
* PBL
* MCB
* JS Bank

---

### 4. Account Info

Contains customer account details:

* Account Number
* Account Type
* Bank ID
* Bank Name
* Branch Name
* Total Deposit Amount
* Total Withdraw Amount
* Available Balance

#### Account Types

* Current Account
* Saving Account

---

### 5. Deposit Info

Maintains deposit transaction records.

#### Fields

* Deposit ID
* Deposit Date
* Account Number
* Account Type
* Bank ID
* Bank Name
* Branch Name
* Deposit Amount
* Mode of Payment

#### Payment Methods

* Cash
* Online
* Cheque

---

### 6. Withdraw Info

Stores withdrawal transaction records.

#### Fields

* Withdraw ID
* Withdraw Date
* Account Number
* Account Type
* Bank ID
* Bank Name
* Branch Name
* Withdraw Amount
* Mode of Payment

#### Payment Methods

* Cash
* Online
* Cheque

---

### 7. Report of Account Info

Generates summaries of customer accounts and balances.

### 8. Report of Deposit

Provides reports for all deposit transactions.

### 9. Report of Withdraw

Provides reports for all withdrawal transactions.

The **Final Project of Bank Management System 2018** is a simple yet effective banking information management system developed using Microsoft Excel.
